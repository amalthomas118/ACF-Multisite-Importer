== Description ==

**Multisite ACF Importer** is a powerful plugin designed to streamline the process of importing Advanced Custom Fields (ACF) settings across a WordPress multisite network. This plugin allows you to easily import ACF field groups into multiple sites within your network, saving you time and ensuring consistency across all sites.

== Short Description ==

**The MSAI Plugin (Multi Site ACF Importer) plugin is a plugin which can be used to import ACF fields to multiple sites at once, making it easier than manually uploading the JSON file to multiple websites.

### Features
- Import ACF field groups across a multisite network
- Easy-to-use interface for managing imports
- Supports ACF JSON export files
- Ensures consistent ACF settings across all sites

== Installation ==

1. Upload the `multisite-acf-importer` folder to the `/wp-content/plugins/` directory.
2. Activate the plugin through the 'Plugins' menu in WordPress.
3. Navigate to the `Multisite ACF Importer` menu item in the Network Admin dashboard.
4. Use the interface to select and import your ACF JSON files.

== Frequently Asked Questions ==

= What is required to use this plugin? =

You need to have the Advanced Custom Fields (ACF) plugin installed and activated across your multisite network.

= Can I use this plugin on a single site installation? =

This plugin is designed specifically for multisite networks. For single site ACF imports, use the standard ACF import functionality.

= What file formats does this plugin support for importing ACF settings? =

The plugin supports ACF JSON export files.
